#### sum求和函数

结构：np.sum(qty,axis=1)

qty：数组名        	axis：压缩维度

作用：二维数组中可实现行或列的求和

```python
import numpy as np
qty = np.arange(9).reshape(3, 3)
sum_ = np.sum(qty, axis=0)		 	# 0 => 行被压缩消失 => 列求和  
print(qty)							# 1 => 列被压缩消失 => 行求和
print(sum_) 




# 三维数组：shape=(2, 3, 4) → 2个块、3行、4列
qty_3d = np.arange(24).reshape(2, 3, 4) 	# 三维，所以axis值可为0/1/2
sum_3d = np.sum(qty_3d, axis=0)		 	#  (2, 4) → 压缩了第一维
print(qty_3d)
print(sum_3d) 

sum_3d = np.sum(qty_3d, axis=2)			#  (2, 3) → 压缩了第三维
```
