查看数据的类型：数据类型的关键字  =>  type(数据/变量名)

# 数据类型

## 可变数据类型

### 列表    => list()

### 集合    => set()

### 字典    => dict()

## 不可变数据类型

### 数值类型

#### 整数型 => int

```python
num = 100
print(type(num))  # 函数的执行:从里向外顺序执行   =>  <class 'int'>
```

#### 浮点型(小数) => float

```python
num = 100.1
print(type(num))    =>  <class 'float'>
```

#### 布尔型 => bool => 特殊的数值型
布尔型:True/False -->python中的关键字

True代表的特殊数字是1；False代表的特殊数字是0

注意点：首字母一定是大写

```python
print(True)
print(False)
print(True+1)   	# 1+1==2
print(False+1)  	# 0+1==1
print(type(True))   # => <class 'bool'>
print(type(False))  # => <class 'bool'>
print(true)  # 报错，未定义的变量，而不是布尔值
print(false)  # 报错，未定义的变量，而不是布尔值
```




#### 复数 =>complex  (了解)
格式：a+bj

复数由：实部a+虚部b(j/J)组成

```python
com = 5+6j
print(com)  # (5+6j)
print(type(com))  # <class 'complex'>
```



### 字符串 =>str
不是数值类型

所有被引号包围的数据都是字符串

```python
print("字符串")
print("100")
print("100.1")
print("5+6j")
print("True")
print(type("name"))  # => <class 'str'>
```
