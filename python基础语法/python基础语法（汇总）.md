# 07_运算符——

##   1.比较运算符[关系运算符]

###  1.1 作用

用于比较两个值或表达式之间的关系，并根据比较结果返回布尔值True或False。



### 1.2 分类

1. ==(等于)：如果两边的值相等，则返回True；否则返回False。

1. !=(不等于)：如果两边的值不相等，则返回True；否则返回False。

```python
print(1 == 1)     # => True
print(1 != 1)     # => False
print(3 == 5)     # => False
print(3 != 5)     # => True
print(1.0 == 1)   # => True
print(1 == "1")   # => False  数值类型和字符串无法比较
print("bingbing" == "bingbing")   # => True
print("a" == "b")                 # => False
```



1. (大于) >：如果左边的值大于右边的值，则返回True；否则返回False。

```python
print(3 > 5)          # => False
print(5 > 3)          # => True
print(3 > "5")        # => TypeError: '>' not supported between instances of 'int' and 'str'
print("3" > "5")      # => False
print("33" > "325")   # => True  字符串比较是根据首字符ascii编码值
```



1. (大于等于) >=：如果左边的值大于或右边的值，则返回True；否则返回False。

```python
print(3 >= 3)         # => True
print(5 >= 8)         # => False
```



1. <(小于)：如果左边的值小于右边的值，则返回True；否则返回False。

   与 > 同理

1. <=(小于等于)：如果左边的值小于或等于右边的值，则返回True；否则返回False。

```python
print(3 <= 5)         # => True
print(5 < 3)          # => False
print(3 <= 3)         # => True
```



##  2 逻辑运算符      多条件判断

###  2.1 作用

通过使用逻辑运算符，开发者可以将多个条件表达式组合起来，形成更复杂的条件判断语句。

不仅能够提高代码的可读性，还使得程序的逻辑结构更加清晰、易于理解和维护。

此外，它们还支持短路行为，这在一定程度上提高了程序的执行效率，避免了不必要的计算或操作。



### 2.2 分类

1. and(与、且)：当所有条件都为True时，整个表达式的结果才为True；否则，结果为False。

```python
print(True and True)      # True
print(True and False)     # False
print(False and True)     # False
print(False and False)    # False
# 需求：定义三个变量x、y、z，并检查这三个变量是否都为正数，满足条件则输出语句。
# 条件：x > 0   y > 0   z > 0   ==>  所有条件都必须成立  ==>  and连接
x = 5
y = 10
z = 15
# and：确保所有条件都必须为True
if x > 0 and y > 0 and z > 0:   # True and True and True
print("x，y，z都为正数~")			# 如果条件为真(即x,y,z都为正数)，则执行这个代码块
```



1. or(或)：只要其中一个条件为True，整个表达式的结果就为True；否则结果为False。

```python
print(True or True)      # True
print(True or False)     # True
print(False or True)     # True
print(False or False)    # False
# 需求：定义三个变量x，y，z，并检查这三个变量中是否有一个是正数，满足条件则输出语句。
x, y, z = -5, -10, 15
# 检查变量x,y,z是否至少有一个是正数
# or：确保只要有一个条件为True，整个表达式的结果就为True
if x > 0 or y > 0 or z > 0:
# 如果条件为真(即x,y,z中至少有一个数是正数)，则执行这个代码块
print("至少有一个数大于0~")
```



1. not(非，否)：用于反转条件的结果。如果条件为True，则结果为False；如果条件为False，则结果为True。

```python
print(not True)    # False
print(not False)   # True
# 需求：定义一个变量x，检查它是否不等于10.
x = 5
if not x == 10:   # not False  ==> True
print("x不等于10~")
# not x == 10 等效于 x != 10
# 注意：在Python中，一个数据既不为零(包括任何形式上的零，如0、0.0、False、0j)，也不为空(包括任何形式上的空，如""、[]、()、{}、set()、None等)，那么它被用作条件表达式时，Python会将其视为True。
# 非零非空即为True。
print(not 1)    # False
print(not 0)    # True
print(not -10000000)   # False
print(not 0.0)         # True
print(not 10.0)        # False
print(not "")          # True
print(not " ")  	   # False
print(not [])		   # True
print(not [ ])		   # True
print(not "bingbing")  # False
```





###  2.3 短路行为【拓展】

and：如果第一个操作数据为False，则整个表达式的结果立即确定为False，Python不会判断第二个操作数据。

or：如果第一个操作数据为True，则整个表达式的结果立即确定为True，Python不会判断第二个操作数据。

```python
print(3 and 5)   # 3(True) and 5(True)
print(5 and 3)   # 5(True) and 3(True)
print(0 and 3)   # 0(False) and
print(3 and 0)   # 3(True) and 0(False)
print(0 or 0)   # 0(False) or 0(False)
print(0 or 5)   # 0(False) or 5(True)
print(5 or 0)   # 5(True) or
print(3 or 5)   # 3(True) or
```

## 3. 三元表达式

### 3.1 作用

通过允许在一行代码中完成条件判断和值的选择,显著的减少了需要编写的代码量,这种简洁性使得代码更加紧凑,减少了冗余,并且提高了可读性.当处理简单的条件逻辑时,使用三元表达式可以使代码更加清晰明了.



### 3.2 格式

为真结果 if 条件 else 为假结果



应用:检查一个人的年龄并根据其是否成年来输出相应的消息

```python

```

注意:当条件逻辑复杂\需要执行多个语句\嵌套的三元表达式时,更推荐使用if-else或其他结构.





## 4. if-elif    (多选一)

### 4.1 格式

if 条件表达式1:
​    条件1成立时执行的代码块
elif 条件表达式2:
​    条件2成立时执行的代码块
elif 条件表达式3:
​    条件3成立时执行的代码块
...  		# elif没有数量限制,可以根据需要添加多个elif语句来检查多个条件
else:	#else可以不用，但若上述条件中都未通过则什么都不会输出
​    所有条件都不成立时执行的代码块



应用:检查天气情况,并根据结果输出相应的穿衣建议

```python
# # 定义变量保存天气情况
weather = input("请输入天气:")
if weather == "晴天":
# 如果天气为"晴天",则执行这个代码块
   	print("今天阳光明媚,建议穿短袖和短裤.")
elif weather == "多云天":
# 如果天气为"多云天",则执行这个代码块
	print("今天多云,建议穿轻便的外套和长裤.")
elif weather == "雨天":
# 如果天气为"雨天",则执行这个代码块
   	print("今天下雨,请记得带伞,并穿上防水的外套和鞋子.")
else:
# 如果天气不是上述中的任何一种,则执行代码块
	print("今天的天气情况不在预设范围之内,请根据实际情况选择合适的衣物.")
```



### 4.2 注意

1. if-elif-else结构按顺序在多个条件中逐一检查，一旦某个条件满足(即为True)，则立即执行该条件下对应的代码块，并跳过所有后续的 elif 和 else 及其对应的代码块。

1. 如果所有的条件都不满足,则直接执行else下的代码块.

1. if多分支，if一定是第一个分支,后续其它分支为elif

1. else不是必须的，若有则必须放在最后

1. 如果多分支中，同时满足多个判断条件，执行第一个判断的结果，不会继续向后执行

## for和while循环的区别

1. 循环次数：
   for循环有固定的循环次数，常用于遍历可迭代对象
   while循环常用在循环次数不确定的场景中

1. 终止条件：
   for循环将可迭代对象中的元素取完之后，循环结束
   while循环需要判断条件来决定循环什么时候结束

1. 无限循环
   for循环不存在无限循环
   while存在无限循环







## break和continue

### 1. break

作用：结束循环
应用：从苹果篮子中取苹果吃，吃到坏的苹果，终止吃苹果的动作

```python
for i in range(1, 100):
# 将被五整除的数字，设置成坏苹果
	if i % 5 == 0:
        break
    print(f"吃苹果，第{i}个苹果")
```

### 2. continue

作用：跳过某次循环，继续下一次循环
应用：从苹果篮子中取苹果吃，吃到坏的苹果，把坏苹果丢掉，继续吃后面的苹果

```python
for i in range(1, 100):
# 将被五整除的数字，设置成坏苹果
    if i % 5 == 0:
        continue
    print(f"吃苹果，第{i}个苹果")
```
