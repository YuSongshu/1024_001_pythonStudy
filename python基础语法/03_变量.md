# 03_标识符及变量



## 1. 标识符

### 1.1 概念：用来标识变量、函数、类等的名称

​		变量名是标识符的一种

### 1.2 组成：

​	数字、字母、下划线组成

### 1.3 开头：

​	只能由字母、下划线，不能由数字开头

### 1.4 要求：

#### 	1.4.1 不能使用python的关键字作为变量名

​		可以使用*keyword*模块查看当前版本的所有关键字：​	

```python
import keyword
import builtins
print(keyword.kwlist)		# 查看python中常见的关键字
print(dir(builtins))		# 查看python中所有的关键字
```

​		原因：

​			将关键字作为变量名，会覆盖掉关键字的原有功能

​			print()-->关键字-->作为变量名-->新数据覆盖旧数据-->print()函数原有的功能会消失

​			print = "hello world!"

​			*print(print)*

>
> ​		**运算符关键字**：如*and*、*or*、*not*、*is*、*in*，用于逻辑和对象操作。
>
> ​		**控制流关键字**：如*if*、*elif*、*else*、*for*、*while*、*break*、*continue*，用于控制程序执行流

> ​		**异常处理关键字**：如*try*、*except*、*finally*、*raise*，用于捕获和处理异常。

> ​		**模块与作用域关键字**：如*import*、*from*、*as*、*global*、*nonlocal*，用于模块管理和变量

#### 	1.4.2 严格区分大小写

​		虽然单词的意思一样，但是不同的变量名

​		name = "张三"
​		Name = "李四"
​		namE = "王五"
​		print(name, Name, namE)



### 1.5 命名规范：下划线命名法(变量名)、大驼峰命名法、小驼峰命名法

​	下划线命名法(变量名)：单词之间，用下划线分隔开

​		student_score = 88

​	大驼峰命名法；每个单词的第一个字母大写的，其它字母小写

​		StudentScore = 88

​	小驼峰命名法：除第一个单词全小写外，其它单词的第一个字母还是大写

​		studentScore = 88



## 2.变量

​	概念：是一个容器，计算机中的存储空间

### 2.1 作用
​	临时保存运行的数据，在不同的代码中传递，记录程序运行的状态

### 2.2 变量的定义
创建变量：变量名 = 数据

使用数据，通过变量名

```python
money = 100
print(money)
```

通过变量名，可以记录程序运行的值

```python
step1 = 100 + 1  # 第一步的数据
print(step1)		# => 101
step2 = step1 * 2  # 第二步的数据
print(step2)		# => 202
# 个人信息：姓名、年龄、身高
name = '小王'
age = 18
height = 177
print(name)  # => 小王
print(age)  # => 18
```



### 1.3 一个变量被多次赋值
原则：新数据覆盖旧数据

```python
age = 21
age = 18	# 会标黄但不会报错，不规范
print(age)  # => 18
```



### 2.4 变量的使用原则：先定义后使用
```python
money = 10000	# 变量的定义
print(money)
print(weight)  # 使用weight变量名前，变量不存在，会报错
```



### 2.5 变量定义的规范性：

做到见名知意-->变量名和变量中存储的数据有关联性

​	1.	不规范的使用

​		home = "苹果"

​		正确的规范性使用

​		apple = "苹果"

​	2.组成：数字、字母、下划线组成

​		name = "张三"

​		name_ = "张三"

​		name2 = "张三"

​	3.开头：只能由字母、下划线，不能由数字开头

```python
# 2name = "张三"   =>数字放在变量的开头，报错
_name = "张三"
print(_name)
```

### 2.6 注意点

变量使用时，不加引号！！！！！！

被引号包围的都是字符串，不是变量名

```python
money = 10000
print(money)  		# 打印变量名，不加引号
print("money")  	# 打印字符串，被引号包围的都是字符串
```
