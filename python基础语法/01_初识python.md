# 1.编程语言类型的了解

编译器-->编译型语言

C是编译型语言-->一次编译之后，多次使用

解释器-->解释性语言

python解释器-->运行编写好的python代码-->python.exe

python动态解释型语言-->每一次运行，都要重新解释

源代码：写好的python代码

```python
print('hello world')  
```



高级语言-->【编译器】-->二进制数据  --> 编译型语言

高级语言-->【python解释器】-->二进制数据  --> 解释性语言

编译型语言&解释性语言

1. 编译型语言：提前把数据处理好，多次运行，速度更快

2. 解释型语言：运行时处理数据，速度慢，跨平台性更好

编程语言，在不同的操作系统中，编译方式是不同的



# 2.编写第一个python代码

## 2.1 编写代码

​	(1)新建python文件：鼠标右键文件夹的位置-->new-->python file -->输入文件名-->回车确认

​	(2)编写代码：确保代码没有红色波浪线的错误提示

​	(3)代码的运行：鼠标右键，代码空白区域-->Run-->点击运行

​	(4)查看代码运行的结果：在下方的控制台

​			内容打印时，数字不加引号，字符串加引号

```python
print(100)
print(0)
print('hello world')
```



# 3.代码注释

## 3.1 单行注释

​	使用# 注释一行内容

```python
#注释一行内容
```



## 3.2 多行注释

三对引号(单引号、双引号)

```python
'''
多行注释的内容
注释的第二行
注释的第三行
'''

"""
多行注释的内容
注释的第一行
注释的第二行

"""
```




# 4.报错提示

## 4.1 编写代码时的错误提示

红色波浪线：语法错误，运行时代码一定会报错

白色波浪线：规范使用不规范，运行代码不报错

绿色的波浪线：对拼写的单词进行检查，说明单词拼写可能错了

黄色高亮：警告信息，代码可能存在问题，提醒你对这个可能存在问题的代码进行优化

## 4.2 代码运行完给出的错误提示

```python
print(python)    #会有红色标线
```

给出报错点所在的行数，错误的类型，错误提示信息

## 4.3 python代码的规范性要求：PEP8

​	单行注释后推荐加一个空格

​	使用print()函数时，数据和数据之间的逗号后，推荐加一个空格

​	print('hello wrold') 	# 英文拼写错误

​	print('pingyin')  	# 将中文拼写检测成单词


# 5.快捷键的使用

​	ctrl + c: 复制的快捷键

​	ctrl + v: 粘贴的快捷键

​	ctrl + d：快速复制上一行的内容

​	ctrl + /: 对多行内容进行快速注释

​	ctrl + alt + L: 自动规范化代码格式

​	ctrl + z: 撤销上一步的操作

​	ctrl + y: 恢复撤销
