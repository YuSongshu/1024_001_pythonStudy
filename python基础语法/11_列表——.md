# 11_列表——

**定义**：由**有限个元素**组成的**有序序列**，元素按插入顺序排列，可通过**索引**访问。

作用:一次性存储多个数据，而且可以为不同数据类型

格式：变量名=[元素1， 元素2，元素3...]

```
 lst = [21, 21.003, 'hello', 5+2j, True]
 print(lst, type(lst))
```

输出：

```
[21, 21.003, 'hello', (5+2j), True, ['python', 65]] <class 'list'>
```

## 列表的添加(增)

`insert()`   `append() ` ` extend() `  `+`

### insert()--插入

格式： 列表对象.insert(索引位置, 要插入的元素)

原位置及后面的元素自动向后顺延，列表长度 + 1

支持正索引（从前往后）、负索引（从后往前）

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
lst.insert(0, '这是一个列表')
print(lst)
```

输出：

```
['这是一个列表', 21, 21.003, 'hello', (5+2j), True, ['python', 65]]
```



```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
lst.insert(-1, '这是一个列表')
print(lst)
```

输出：

```
[21, 21.003, 'hello', (5+2j), True, '这是一个列表', ['python', 65]]
```

**注：**

1.insert() 方法没有返回值（返回 None），所以打印结果为 None，但列表本身会被修改

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
print(lst.insert(-1, '这是一个列表'))
print(lst)
```

输出：

```
None
[21, 21.003, 'hello', (5+2j), True, '这是一个列表', ['python', 65]]
```



2.添加元素至最后一位

```
lst.insert(len(lst), 元素)
```

索引值大于长度不会报错而是添加元素至最后一位，负数同理（不会报索引越界错误）





### append()--元素追加到列表的尾部

格式：列表名.append(元素值/变量)

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
new_lst = ['学习', '写作业']
lst.append('这是一个列表')
#lst.append('学习', '写作业')  # 多元素写入，报错，存放变量再进行追加，但添加的是列表
lst.append(new_lst)
print(lst)
```

输出：

```
[21, 21.003, 'hello', (5+2j), True, ['python', 65], '这是一个列表', ['学习', '写作业']]
```



### extend()--字符串/列表

格式：列表.extend(可迭代对象)

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
new_lst = ['学习', '写作业']  # 列表的遍历根据索引值
lst.extend(new_lst)
lst.extend('hello')  # 字符串作为可迭代对象，每迭代一次，取一个字母存放到列表
print(lst)
```

输出：

```
[21, 21.003, 'hello', (5+2j), True, ['python', 65], '学习', '写作业', 'h', 'e', 'l', 'l', 'o']

```



### +拼接--相同数据类型进行拼接

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
new_lst = ['hello']

print(lst + new_lst)
```

输出：

```
[21, 21.003, 'hello', (5+2j), True, ['python', 65], 'hello', '9']
```



### 总结

insert(索引值，插入的元素值)--不能在列表的尾部进行插入

append(追加的元素值)--只能在列表的尾部进行追加，一个参数

extend(迭代对象)--字符串（迭代时把每个字母拆分成元素插入），列表（通过索引值遍历）

+拼接 碧玺相同数据类型才可进行拼接



## 列表的删除(删)

```
pop()   remove()   del     clear()
```

### pop(索引值)

默认删除列表最后一个元素

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
lst.pop(1)
lst.pop()
print(lst)
```

输出：

```
[21, 'hello', (5+2j), True]
```

注：

**按索引删除**，如果索引不存在，会报**索引越界 **（IndexError）



### remove(元素值)

元素移除

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
lst.remove(21.003)
print(lst)
```

输出：

```
[21, 'hello', (5+2j), True, ['python', 65]]
```



### clear()--清除列表所有元素

```
lst = [21, 21.003, 'hello', 5+2j, True, ['python', 65]]
lst.clear()
print(lst, type(lst))
```

输出：

```
[] <class 'list'>
```



### 总结

list.pop(索引值)
list.remove(元素值)
del list[索引值]
list.clear() 清除列表中所有元素
强调点--删除时，列表索引值变化的（复制新列表遍历/倒序的遍历）



### 应用

删除列表里的偶数

方法1：复制一个新列表，新列表作为遍历对象

```
list_num = [1, 7, 9, 8, 12, 33, 1]
list_num2 = [1, 7, 9, 8, 12, 33, 1]
for value in list_num2:
    if value % 2 == 0:
        list_num.remove(value)
print(list_num)
```

### 注

for循环通过索引值遍历，删除时列表长度变化，所以pop(索引值)删除时要注意

```
# 目标：删除所有等于 3 的元素
nums = [1, 2, 3, 3, 5]
for i in range(len(nums)):
    if nums[i] == 3:
        nums.pop(i)  # 删除后列表变短，后续索引错位

print(nums) 	#会报索引越界（IndexError）
```

方法1：复制一个新列表，新列表作为遍历对象

```
nums = [1, 2, 3, 3, 5]
new_nums = [1, 2, 3, 3, 5]
for i in range(len(new_nums)):
    if new_nums[i] == 3:
        idx = nums.index(item) 	# 找到当前 item 在原列表中的索引并删除
        nums.pop(i) 
print(nums) 
```

方法2：逆序输出

```
list_num = [1, 2, 3, 3, 5]
for i in range(len(list_num)-1, -1, -1):
    if list_num[i] == 3:
        del list_num[i]
print(list_num)
```

```
不会漏删（核心原理）
正序遍历删除：删除一个元素后，后面元素向前移动，下一轮循环会跳过补位上来的元素（容易漏删）。
倒序遍历删除：从后往前删，删除后面元素只会影响已遍历过的索引，未遍历的前面索引完全不受影响，所有元素都能被检查到
```

## 排序

sort()   sorted()    reverse()

### sort(key=选择属性进行排序,reverse=排序先后)

reverse默认等于False，从小到大排序，可选参数

```
users = [
    {"name": "张三", "age": 20},
    {"name": "李四", "age": 18},
    {"name": "王五", "age": 22}
]

# 按 age 升序（默认）
users.sort(key=lambda x: x["age"])
print(users)
# 按 age 降序
users.sort(key=lambda x: x["age"], reverse=True)
print(users)

#list.sort() 是原地排序，返回 None，所以直接 print(users.sort(...)) 只会输出 None
print(users.sort(key=lambda x: x["age"], reverse=True)) 

```

输出：

```
[{'name': '李四', 'age': 18}, {'name': '张三', 'age': 20}, {'name': '王五', 'age': 22}]
[{'name': '王五', 'age': 22}, {'name': '张三', 'age': 20}, {'name': '李四', 'age': 18}]
None
```



### sorted(可迭代对象, key=排序函数, reverse=False)

新列表 = sorted(可迭代对象, key=排序函数, reverse=False)

```
users = [    {"name": "张三", "age": 20},    {"name": "李四", "age": 18},    {"name": "王五", "age": 22} ] 
# 按 age 升序（默认） 
users_sorted = sorted(users, key=lambda x: x["age"]) 
# 按 age 降序 
users_sorted_desc = sorted(users, key=lambda x: x["age"], reverse=True)
```



`sort()`与 `sorted()`区别

`sort()`（列表方法）和 `sorted()`（内置函数）

|               | 原地修改 | 返回值 | 适用对象       |
| ------------- | -------- | ------ | -------------- |
| `list.sort()` | ✅ 是     | `None` | 仅列表         |
| `sorted()`    | ❌ 否     | 新列表 | 所有可迭代对象 |



### 拓展

#### key：按 “属性 / 字段” 排序

作用：传入**一元函数**，从每个元素中提取排序依据（键）

常用写法：

内置函数：`len`（按长度）、`str.lower`（不区分大小写）。

`lambda` 匿名函数：取元素的**索引 / 键 / 属性**。

`operator.itemgetter`/`attrgetter`：更高效的批量取值。

按属性排序

#### 字典列表：按字典键排序

```
users = [
    {"name": "张三", "age": 20},
    {"name": "李四", "age": 18},
    {"name": "王五", "age": 22}
]

# 按 age 升序（默认）
users_sorted = sorted(users, key=lambda x: x["age"])
# 按 age 降序
users_sorted_desc = sorted(users, key=lambda x: x["age"], reverse=True)
```

#### 自定义对象：按对象属性排序

```
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

students = [Student("小明", 85), Student("小红", 92), Student("小刚", 78)]

# 按 score 升序
sorted_stu = sorted(students, key=lambda s: s.score)
# 按 score 降序
sorted_stu_desc = sorted(students, key=lambda s: s.score, reverse=True)
```

####  元组列表：按元组索引排序

```
data = [("苹果", 5), ("香蕉", 3), ("橙子", 8)]
# 按价格（索引1）升序
data_sorted = sorted(data, key=lambda x: x[1])
# 按价格降序
data_sorted_desc = sorted(data, key=lambda x: x[1], reverse=True)
```

#### 字符串：按长度 / 不区分大小写排序

```
words = ["banana", "Apple", "cherry", "date"]
# 按长度升序
print(sorted(words, key=len))  # ['date', 'Apple', 'banana', 'cherry']
# 不区分大小写升序
print(sorted(words, key=str.lower))  # ['Apple', 'banana', 'cherry', 'date']
```
