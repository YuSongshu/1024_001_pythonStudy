# 13_字典

字典是一种可变数据  <class 'dict'>

格式：dict={key1:value1, key2:value2}

## 特点：
键值对存储

key的唯一性，不能重复，否则会进行数据的覆盖

key的数据类型必须是不可变

示例

```
dct = {'name': '张三', 'age': 18, 'sex': '男'}
dct = {'name': '张三', 'age': 18, 'age': '男'}  #key重复时，后面的键值对，覆盖前面的键值对
dct = {'name': '张三', 'age': 18, ['sex']: '男'}  # TypeError: unhashable type: 'list'
```



## 定义空字典
1.直接定义一个空的花括号，表示为一个空字典

```
dic = {}
```



2.利用函数创建空字典

```
dic2 = dict()
```



注：

为零为空的数据：整数零、浮点数零、复数零、空字符串、空列表、空元组、空字典、空集合、None

## 应用

通过字典存储学生信息，通过学生姓名，输出学生的所有个人信息

```
dct = {
'张三': [18, '男', 177.5],
'李四': [22, '女', 180.5],
'王五': [18, '男', 170.5]
}
```




## 字典的增删改查
### 查
字典的元素查找，是根据字典的key，不能根据索引值查找

#### 1.字典.get(key)

作用：通过key查找字典的value，查找的key不存在，返回None

```
dct = {
'张三': [18, '男', 177.5],
'李四': [22, '女', 180.5],
'王五': [18, '男', 170.5]
}
print(dct.get('张三'))
print(dct.get('李华'))  # None
```

输出

```
[18, '男', 177.5]
None
```



#### 2.字典[key]

语法格式：字典[key] = value

作用：查找的key不存在，提示key不存在，代码报错

```
dct = {
'张三': [18, '男', 177.5],
'李四': [22, '女', 180.5],
'王五': [18, '男', 170.5]
}
print(dct['张三'])
print(dct['张三'][0])  # 18
# print(dct['李华'])  # KeyError: '李华'
```

输出

```
[18, '男', 177.5]
18
```



### 改

#### 1.字典[key] = 新数据
作用：通过key改字典中的值，用新数据覆盖旧数据

```
dct = {
'张三': [18, '男', 177.5],
'李四': [22, '女', 180.5],
'王五': [18, '男', 170.5]
}
dct['王五'] = [18, '男', 172.5]
dct['李四'][-1] = 172.7
print(dct)
```

输出

```
{'张三': [18, '男', 177.5], '李四': [22, '女', 172.7], '王五': [18, '男', 172.5]}

```





### 删

#### 1.del 字典[key]

注：

如果删除的key不存在，报错KeyError
如果删除整个字典，变量的引用也被删除

```
dct = {
 '张三': [18, '男', 177.5],
 '李四': [22, '女', 180.5],
 '王五': [18, '男', 170.5]
}
del dct['王五']
# del dct  # 删除的是变量名，变量名被删除后，变量名不存在，输出会报错
print(dct)
```

输出

```
{'张三': [18, '男', 177.5], '李四': [22, '女', 180.5]}
```





#### 2.字典.pop(key) 

注：指定key删除字典中的键值对，如果删除的key不存在则报错

```
dct = {
 '张三': [18, '男', 177.5],
 '李四': [22, '女', 180.5],
 '王五': [18, '男', 170.5]
}
dct.pop('王五')
# dct.pop('王五')  # 重复删除，删除的key不存在，KeyError: '王五'
print(dct)
```



#### 3.字典.clear()

作用：清除所有的键值对

```
dct = {
 '张三': [18, '男', 177.5],
 '李四': [22, '女', 180.5],
 '王五': [18, '男', 170.5]
}
dct.clear()
print(dct)

```

输出

```
{}
```



### 增

#### 1.字典[新key] = 数据

作用：在字典中添加新的键值对

```
dct = {
 '张三': [18, '男', 177.5],
 '李四': [22, '女', 180.5],
 '王五': [18, '男', 170.5]
}
dct['李华'] = [33, '女', 170.5]  # 一次添加一个新的键值对
print(dct)
```

输出

```
{'张三': [18, '男', 177.5], '李四': [22, '女', 180.5], '王五': [18, '男', 170.5], '李华': [33, '女', 170.5]}

```



#### 2.字典.update(可迭代对象)

对于可迭代对象，必须包含二级元素，单个元素的长度是2，可以进行元素的拆分--key和value
可迭代对象：列表、元组、字典、集合
注：字符串不支持

```
dct = {
 '张三': [18, '男', 177.5],
 '李四': [22, '女', 180.5],
 '王五': [18, '男', 170.5]
}
lst = [["李华", [33, '女', 170.5]], ["小明", [26, '男', 188]]]
lst = [['a', 1], ['b', 2], ['c', 3]]
tu = (('a', 1), ('b', 2), ('c', 3))

dct2 = {'a': 1, 'b': 2, 'c': 3}
dct.update(tu)
print(dct)
```

 直接修改原字典，不会返回新字典

其也可以用于更新（改）数据

```
dct = {"name": "小明", "age": 20}
dct.update(age=18)
print(dct)
```



## 基础操作

### 字典的for循环取值

```
dct = {
 12: 'a',
 13: 'b',
 15.66: 'c',
 'd': 'd',
 199.5: 'e',
}
new_dct = {}
for i in dct:
    if isinstance(i, int):
        new_dct[i] = dct[i]

print(new_dct)
```

输出

```
{12: 'a', 13: 'b'}
```



### 字典.keys()
作用：取出字典中所有的key

```
dct = {
    12: 'a',
    13: 'b',
    15.66: 'c',
    'd': 'd',
    199.5: 'e',
}
k = dct.keys()

print(k, type(k))  # <class 'dict_keys'>  既不是元组，也不是列表，是字典中的一种特殊数据类型，可迭代对象
```

输出

```
dict_keys([12, 13, 15.66, 'd', 199.5]) <class 'dict_keys'>
```



### 字典.values()

作用：取出字典中所有的value

```
dct = {
    12: 'a',
    13: 'b',
    15.66: 'c',
    'd': 'd',
    199.5: 'e',
}
v = dct.values()
print(v, type(v))  


```

输出

```
dict_values(['a', 'b', 'c', 'd', 'e']) <class 'dict_values'>
```



### 字典.items()

作用：取出字典中所有的键值对

```
dct = {
    12: 'a',
    13: 'b',
    15.66: 'c',
    'd': 'd',
    199.5: 'e',
}

res = dct.items()
print(res, type(res))
```

输出

```
dict_items([(12, 'a'), (13, 'b'), (15.66, 'c'), ('d', 'd'), (199.5, 'e')]) <class 'dict_items'>

```

dict_items([(12, 'a'), (13, 'b'), (15.66, 'c'), ('d', 'd'), (199.5, 'e')])
每个元组元素的长度，为2-->key:value



意义

元组可以解包

```
dct = {
    12: 'a',
    13: 'b',
    15.66: 'c',
    'd': 'd',
    199.5: 'e',
}
for i in dct.items():  # 临时变量i中存放的是元组-->元组长度是2
    print(f'{i}')
# (12, 'a')--> 解包--> k:12, v:'a'
# (13, 'b')--> 解包--> k:13, v:'b'
for k, v in dct.items():
    print(f'{k}:{v}')
```

输出

```
(12, 'a')
(13, 'b')
(15.66, 'c')
('d', 'd')
(199.5, 'e')
12:a
13:b
15.66:c
d:d
199.5:e

```

### 成员运算符  in/not in

作用：判断key有没有在字典中

```
dct = {
    12: 'a',
    13: 'b',
    15.66: 'c',
    'd': 'd',
    199.5: 'e',
}

print('d' in dct)
print('d' not in dct)
```

输出

```
True
False
```
