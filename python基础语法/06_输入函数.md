# 06_输入函数——





## 1.含义

在Python中，程序获取用户输入数据的功能通常被称为输入操作，这一功能是通过内置函数 input 来实现



## 2.语法

变量名=input('提示信息') 		#提示信息，会在控制台显示，用来提示用户输入什么样的数据，可省略

```python
name=input('请输入你的名字')
```



注：

1.输入数据之后，回车确定，代码才会继续向下执行

2.提示信息不会写入到变量中

3.输入函数输入的数据都是字符串类型

```python
a = input()
print(a , type(info))
```



## 3.数据类型转换

代码由内向外执行

### 3.1 在input函数外进行转换

```python
num=int(input('数字：')) 
```



### 3.2 在使用函数数据时再进行转换

```python
num=input('数字：')
print(int(num))
```



### 3.3 注意

1.float转换为int 取整数部分不会四舍五入

2.使用float函数时就算是整数也会带小数部分

3.计算数据精度不一致时，精度以精度更高的数据为准

4.浮点型运算时会精度缺失，因为将数据转为二进制

5."+"号两边数据类型要一致，否则报错，若无int则只进行字符串的拼接           =>  1+1=11
