## 1.%占位符 

### 1.1 %d ：给整数占位



### 1.2 %f ：给浮点数占位



### 1.3 %s ：给字符串占位，通用的占位符(可以给任意的数据占位)

语法格式： print("提示内容%d 提示信息%f 提示信息%s" % (整数, 浮点数, 字符串数据))

```python
name = "小明"
age = 18
height = 177.6
print("我的名字是：%s，年龄是：%d，身高是：%f" % (name, age, height))
#  => 我的名字是：小明，年龄是：18，身高是：177.600000
```

注:使用%f后，浮点数的精度会扩大，补全到六位小数



### 1.4 数据精度的控制-->四舍五入

格式：print("提示内容%.nf" % 浮点数)

```python
num = 175.254687
print("保留2位小数 %.2f" % num)  # 保留2位小数 175.25
print("保留整数位 %.0f" % 17.984)  # 保留整数位 18
```





## 2.占位符使用不匹配

```python
print("我的名字是：%s，年龄是：%d，身高是：%d" % (name, age, height))

我的名字是：小明，年龄是：18，身高是：177
```

问题？身高的数据把浮点数的小数位，丢掉

```python
print("我的名字是：%d，年龄是：%s，身高是：%s" % (name, age, height))  # 报错
```

问题？数字之间%d和%f可以互用，但存在精度丢失和精度的扩大；如果是字符串，只能用%s，否则会报错

## 4.f表达式

### 4.1 格式：

print(f"提示内容{变量名}")

```python
name = "小明"
age = 18
height = 177.6
print(f"我的名字是：{name}，年龄是：{age}，身高是：{height}")
```

f后，只有一个字符串数据。数据之间需要用逗号分隔开

不存在数据精度的丢失以及精度的扩展



### 4.2 数据精度的控制-->四舍五入

格式：print(f"{变量名:.nf}")

```python
num = 175.929687
print(f"控制精度，保留2位小数：{num:.2f}")
print(f"控制精度，保留整数：{num:.0f}")
```



### 4.3 支持对表达式进行操作

```python
print(f"表达式计算：{num + 10:.1f}")  # 计算完成后的数据保留一位小数
```





## 5.python内置函数，控制数据精度

### 5.1 round(变量名, n)

```python
print(round(num, 2))  # 保留2位小数
```



### 5.2 format(变量名,'.nf')

```python
print(format(num, '.2f'))  # 保留2位小数
print('{:.2f}'.format(num))  # 保留2位小数
```
