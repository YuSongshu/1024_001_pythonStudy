# 14_集合

集合是一个无序不重复的元素序列

格式：set = {元素1, 元素2}

特点：去重性、无序性（）

```
se = {11, 22, 33, 'aa', 12.6, (11, 22), 22, 33}  # 自动去除重复元素
# se2 = {11, 22, 33, 'aa', 12.6, (11, 22), [22, 33]}  # TypeError: unhashable type: 'list'
print(se, type(se))  # <class 'set'>
# print(se[0])  # 集合无下标
```

输出

```
{33, 'aa', 22, 11, 12.6, (11, 22)} <class 'set'>
```



注：

​      使用{}包围数据，但元素没有键值对

​      集合中的元素必须是不可变的数据类型(数值、字符串、元组)

​      集合元素不可变，但集合本身是可以进行修改的



## 集合的增删改查

### 增

#### 集.add(元素)

作用：指定元素，添加到集合中
注：一次增加一个元素

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.add(44)  # 随机添加到某个位置
print(se)
```



#### 集合.update(可迭代对象)

作用：将可迭代对象中的多个元素，添加到集合中

注意点：元素添加没有顺序

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.update([44, 55, 66])
se.update('ACB')
print(se)
```

输出

```
{66, 'aa', 11, 12.6, (11, 22), 22, 33, 'C', 44, 'A', 55, 'B'}
```



### 删

#### 集合.remove(元素)

作用：将集合中的元素移除
注：如果删除的元素不存在，报错

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.remove(33)
se.remove(33)  # KeyError: 33
print(se)
```



#### 集合.discard(元素)

作用：将集合中的元素删除
注：如果删除的元素不存在，不报错

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.discard(33)
se.discard(33)  # 删除的元素不存在，不报错
print(se)
```



#### 集合.pop()

作用：随机删除集合中的元素

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.pop()
print(se)
```



#### 集合.clear()

作用：清除集合中的所有元素

```
se = {11, 22, 33, 'aa', 12.6, (11, 22)}
se.clear()
print(se) 
```

输出

```
set()
```



不是{}而是set()，{}默认是字典

##### 空集合

se = set()

print(se, type(se))  # set() <class 'set'>

se2 = {}

print(se2, type(se2))  # {} <class 'dict'>



### 改

集合不能进行数据的修改，因为找不到旧数据所在的位置，不能用新数据覆盖旧数据

### 查

集合无序，无法查找元素
