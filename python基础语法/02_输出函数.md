# 1.输出函数

```python
print(显示在控制台上的数据,sep="控制数据和数据之间的间隔符", end="控制字符串的结尾字符")
```

作用：将信息显示在控制台上

## 第一个参数：输出的数据

```python
print('使用默认的空格隔开数据')

print()  # 没有输出任何数据

print(11)  # 使用print函数，输出一个数据到控制台

print(11, 22)  # 输出两个数据到控制台

print('hi', 'hello')  # 输出两个数据到控制台
```



​	输出多个数据时，数据和数据之间，用逗号隔开

```python
	print(11, 22, 33)
```



```python
	print(11, 'hi', 33, 'aa')  # 代码中，逗号后空格，规范性的写法，不影响输出内容，只是让代码可读性更行
```



## 第二个参数：sep=" "
sep-->separate：分隔

目的：让输出内容更加美观

如果没有修改sep参数的值，默认使用空格，将数据分开

给sep参数传一个新的值，将数据以指定的符号分隔开

```python
print("使用新的符号-，将数据分隔开")
print(11, 'hi', 33, 'aa', sep='-')		#=>	11-hi-33-aa
print("使用新的符号!，将数据分隔开")
print(11, 'hi', 33, 'aa', sep='!')		#=>	11!hi!33!aa
print("数据和数据之间，没有间隔符")
print(11, 'hi', 33, 'aa', sep='')		#=>	11hi33aa
print("#不是注释内容")  # 放在print函数中，打印在控制台上

#所有被引号的数据，不管引号中是什么，都是字符串数据
```




## 第三个参数：end=”控制字符串的结尾字符“
学习python。  =>结尾字符是句号

学习python！  =>结尾字符是感叹号

学习python+   =>结尾字符是加号

如果没有修改end参数的值，默认使用\n，作为结尾字符

给end参数传一个新的值，将数据以指定的字符作为结尾字符\

### 修改数据的结尾字符
```python
print(11, 'hi', 33, 'aa', end="。\n")  # 数据输出结束后，以"。"作为结尾字符并换行（ 换行符\n在控制台上不显示为\n，对内容进行换行）
print(11, 'hi', 33, 'aa', end="!")  # 数据输出结束后，以"!"作为结尾字符
print(11, 'hi', 33, 'aa')  # 数据输出结果后，进行换行
```

# 3.输出函数使用的注意点
## 3.1 参数使用的注意点
sep=" "间隔符在两个及以上的数据中，使用

sep和end顺序可以变化的，但是必须在所有输出数据的后面

print(11, 'hi', 33, 'aa', sep='-', end='!')
print(11, 'hi', 33, 'aa', end='!', sep='-')

print(11, end='!', 'hi', 33, 'aa')  # 放在数据前，报错

print(11, sep='-', 'hi', 33, 'aa')  # 放在数据前，报错

## 3.2 输出函数规范
1.单个的输出函数顶格写

2.一行只写一个输出函数

3.符号的使用，都使用英文的符号

​	括号、引号使用英文，英文引号中，可以写任意内容

​	中文的引号，报错

4.输出数据加引号

前期的学习，除数字外，其它的字母、中文、特殊符号，输出都要加引号

```python
print('输出函数')
print('hello)’‘’(*&^$')
print(100)
print(100, 'world')
print("100world")
```
