# 15_推导式与集合运算符

## 推导式

### 集合的推导式

格式：{表达式 for i in se if 判断}

```
se = {11, 22, 33, 44, 55, 66}
new_se = {i for i in se if i % 2 == 0}
print(new_se)
```

输出

```
{66, 44, 22}
```



### 字典的推导式

1.通过循环取出key，再通过key找value

```
dct = {11: '张三', 22: '李四', 33: '王五', 44: '赵六'}
new_dct = {i: dct[i] for i in dct if i % 2 == 0}
print(new_dct)
```

输出

```
{22: '李四', 44: '赵六'}
```



2.直接使用items(),把key和value一起取出,对元组(key, value)解包，得到key和value

```
dct = {11: '张三', 22: '李四', 33: '王五', 44: '赵六'}
new_dct2 = {k: v for k, v in dct.items() if k % 2 == 0}
print(new_dct2)
```

输出

```
{22: '李四', 44: '赵六'}
```





## 运算符

交集、并集、差集 原生只支持 set 集合

### 交集  &

作用：两个集合重合的部分，相同点

```
se1 = {1, 2, 3, 4, 5}
se2 = {4, 5, 6, 7, 8}
se3 = {11, 22, 1, 2, 3}
print(se1 & se2) 
```

输出

```
{4, 5}
```



### 差集  -

作用：前一个集合有，而后一个集合没有的部分，不同点

```
se1 = {1, 2, 3, 4, 5}
se2 = {4, 5, 6, 7, 8}
print(se1 - se2)		# 只在se1中出现的元素，而在se2中没有出现
```

输出

```
{1, 2, 3}
```



### 并集  |

作用：两个集合中的组合，并进行去重

```
se1 = {1, 1, 2, 3, 4, 5}
se2 = {4, 5, 6, 7, 8}
print(se1 | se2)  
```

输出

```
{1, 2, 3, 4, 5, 6, 7, 8}
```



### 函数表达

set.intersection() - 求交集

set.difference() - 求差集

set.union() - 求并集





### 注

虽然只支持 set 集合，但可以强制类型转换

```
set([1,2,3])       # 列表→集合
set((1,2,3))       # 元组→集合
set("abc")         # 字符串→集合 {'a','b','c'}
set({"a":1,"b":2}) # 字典→集合，只取键 {'a','b'}
```



拓展：

**list / tuple / set** 三者可互相随意转

字符串、range 全都能转这三种

字典直接转 → 只留**键**；items () 才是键值对

数字不能转集合 / 列表
