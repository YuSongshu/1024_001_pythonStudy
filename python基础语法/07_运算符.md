##   1.比较运算符[关系运算符]
###  1.1 作用
用于比较两个值或表达式之间的关系，并根据比较结果返回布尔值True或False。



### 1.2 分类

1. ==(等于)：如果两边的值相等，则返回True；否则返回False。

2. !=(不等于)：如果两边的值不相等，则返回True；否则返回False。

```python
print(1 == 1)     # => True
print(1 != 1)     # => False
print(3 == 5)     # => False
print(3 != 5)     # => True
print(1.0 == 1)   # => True
print(1 == "1")   # => False  数值类型和字符串无法比较
print("bingbing" == "bingbing")   # => True
print("a" == "b")                 # => False
```



3. (大于) >：如果左边的值大于右边的值，则返回True；否则返回False。

```python
print(3 > 5)          # => False
print(5 > 3)          # => True
print(3 > "5")        # => TypeError: '>' not supported between instances of 'int' and 'str'
print("3" > "5")      # => False
print("33" > "325")   # => True  字符串比较是根据首字符ascii编码值
```



4. (大于等于) >=：如果左边的值大于或右边的值，则返回True；否则返回False。

```python
print(3 >= 3)         # => True
print(5 >= 8)         # => False
```



5. <(小于)：如果左边的值小于右边的值，则返回True；否则返回False。

   与 > 同理

6. <=(小于等于)：如果左边的值小于或等于右边的值，则返回True；否则返回False。

```python
print(3 <= 5)         # => True
print(5 < 3)          # => False
print(3 <= 3)         # => True
```



##  2 逻辑运算符      多条件判断
###  2.1 作用
通过使用逻辑运算符，开发者可以将多个条件表达式组合起来，形成更复杂的条件判断语句。

不仅能够提高代码的可读性，还使得程序的逻辑结构更加清晰、易于理解和维护。

此外，它们还支持短路行为，这在一定程度上提高了程序的执行效率，避免了不必要的计算或操作。



### 2.2 分类

1. and(与、且)：当所有条件都为True时，整个表达式的结果才为True；否则，结果为False。

```python
print(True and True)      # True
print(True and False)     # False
print(False and True)     # False
print(False and False)    # False
# 需求：定义三个变量x、y、z，并检查这三个变量是否都为正数，满足条件则输出语句。
# 条件：x > 0   y > 0   z > 0   ==>  所有条件都必须成立  ==>  and连接
x = 5
y = 10
z = 15
# and：确保所有条件都必须为True
if x > 0 and y > 0 and z > 0:   # True and True and True
print("x，y，z都为正数~")			# 如果条件为真(即x,y,z都为正数)，则执行这个代码块
```



2. or(或)：只要其中一个条件为True，整个表达式的结果就为True；否则结果为False。

```python
print(True or True)      # True
print(True or False)     # True
print(False or True)     # True
print(False or False)    # False
# 需求：定义三个变量x，y，z，并检查这三个变量中是否有一个是正数，满足条件则输出语句。
x, y, z = -5, -10, 15
# 检查变量x,y,z是否至少有一个是正数
# or：确保只要有一个条件为True，整个表达式的结果就为True
if x > 0 or y > 0 or z > 0:
# 如果条件为真(即x,y,z中至少有一个数是正数)，则执行这个代码块
print("至少有一个数大于0~")
```



3. not(非，否)：用于反转条件的结果。如果条件为True，则结果为False；如果条件为False，则结果为True。

```python
print(not True)    # False
print(not False)   # True
# 需求：定义一个变量x，检查它是否不等于10.
x = 5
if not x == 10:   # not False  ==> True
print("x不等于10~")
# not x == 10 等效于 x != 10
# 注意：在Python中，一个数据既不为零(包括任何形式上的零，如0、0.0、False、0j)，也不为空(包括任何形式上的空，如""、[]、()、{}、set()、None等)，那么它被用作条件表达式时，Python会将其视为True。
# 非零非空即为True。
print(not 1)    # False
print(not 0)    # True
print(not -10000000)   # False
print(not 0.0)         # True
print(not 10.0)        # False
print(not "")          # True
print(not " ")  	   # False
print(not [])		   # True
print(not [ ])		   # True
print(not "bingbing")  # False
```





###  2.3 短路行为【拓展】
and：如果第一个操作数据为False，则整个表达式的结果立即确定为False，Python不会判断第二个操作数据。

or：如果第一个操作数据为True，则整个表达式的结果立即确定为True，Python不会判断第二个操作数据。

```python
print(3 and 5)   # 3(True) and 5(True)
print(5 and 3)   # 5(True) and 3(True)
print(0 and 3)   # 0(False) and
print(3 and 0)   # 3(True) and 0(False)
print(0 or 0)   # 0(False) or 0(False)
print(0 or 5)   # 0(False) or 5(True)
print(5 or 0)   # 5(True) or
print(3 or 5)   # 3(True) or
```





## 3. 算术运算符

### 3.1 含义
用于执行基本的数学运算，如加、减、乘、除等。这些运算符直接应用于数值类型上，以执行相应的计算。



### 3.2 分类

#### 加 +  	减 -		乘 *		除 /	  

注：除法结果一般为float，即使答案是整数也会加上一位小数

注意：当使用除法运算符(/)或整除运算符(//)或取余运算符(%)时，必须确保除数(及运算符右侧的数)不能为0，否则会报错ZeroDivisionError



#### 整除 //	

```python
						#10 / 3 = 3.333333...		
print(10 // 3)			# => 3		
						# -10 / 3 = -3.3333...
print(-10 // 3)			# => -4
						# 15 / 6 = 2.5
print(15 // 6)			# => 2
						# 15 / (-6) = -2.5
print(15 // (-6))		# => -3
```



#### 取余数/取模   %

用于返回两个数相除的余数，即整除后剩下的不够除的数。
余数 = 被除数 - 除数 * 取整除结果

```python
# 10 % 3 = 10 - 3 * 3 = 1
print(10 % 3)   	# => 1

# 15 - 5 * 3 = 15 - 15 = 0
print(15 % 5)		# => 0

# 10 % (-3) = 10 - (-3) * (-4) = 10 - 12 = -2
print(10 % (-3))	# => -2
```



#### 幂 **

将第一个数作为底数，第二个数作为指数，进行幂运算

```
print(2 ** 3)   		# 2的3次方 => 2*2*2 => 8
print(3 ** 4)   		# 3的4次方 => 3*3*3*3 => 81
print(1.2e+2)   		# 1.2乘以10的2次方  => 120.0
print(4 ** (1 / 2)) 	# 根号4 	=> 2.0
print(16 ** (1 / 4))	# 四次根号下16 => 2.0
```



注：在Python的算术运算中，只要涉及浮点数，其结果通常也会是浮点数

## 5.赋值运算符

### 5.1 含义

主要用于给变量赋值。在使用时，将右侧的值赋给左侧的变量，右侧也可以在进行某些运算后再赋值给左侧的变量。



### 5.2 分类

#### 1.  简单赋值运算符  =
直接给变量赋值

```python
n = 1
print(n)
```

将一个变量的值赋给另一个变量

```python
n2 = n
print(n2)
```

将运算结果赋值给变量

```python
result = n + n2
print(result)
```



#### 2. 加法赋值运算符 += 

```python
# x += y 等效于 x = x + y
n = 1
n += 1     # n = n + 1 = 1 + 1 = 2
print(n)
m = 1
n = 3
m += n      # m = m + n = 1 + 3 = 4
n += m      # n = n + m = 3 + 4 = 7
m += n      # m = m + n = 4 + 7 = 11
print(m, n)
```

## 6. 转义字符

### 6.1 含义

用于在字符串中表示那些通常具有特殊含义的字符。转义字符以反斜杠\开始，后跟一个或多个字符来表示特定的含义。



### 6.2 常见转义字符

#### 1. \t 制表符

在字符串中插入一个制表符，通常用于对齐文本。   4的倍数

```python
print("bingbinghaha")
print("\tbingbing")
# print("1\t2")            # 1\t  ==>  4-1
# print("11\t2")           # 11\t  ==> 4-2
# print("111\t2")          # 111\t  ==> 4-3
print("1111\t2")         # \t   ==>  4
print("11111\t2")        # 1\t  ==>  4-1
print("111111\t2")
print("1111111\t2")
print("11111111\t2")
print("1\t\t\tbing")
```



#### 2. \n 换行符

在字符串中插入一个换行符，用于在文本中指示新的一行的开始。
print("这是第一行\n这是第二行\n这是第三行")



#### 3.\\\ 反斜杠符号

在字符串中表示一个反斜杠。

```python
print("\\")   # \
print("D:\\tools")
```



#### 4. \\'和\\"  引号

用于在字符串中表示单引号或双引号
print("He said, "Hello World"")    # SyntaxError: invalid syntax. Perhaps you forgot a comma?

内单外双、内双外单

```python
print('It's a beautiful day!')    #报错 SyntaxError: unterminated string literal (detected at line 224)
print("It's a beautiful day!")
```



如果必须使用与字符串界定符相同的引号，可以使用转义字符。

print("He said, \\"Hello World! It's a beautiful day!\\"")
print('He said, "Hello World! It\'s a beautiful day!"')



#### 5. r'' 原始字符串

在Python中，r""表示一个原始字符串。原始字符串不会处理字符串中的转义字符，即字符串中的反斜杠会被当作普通字符处理，而不是转义字符的引导符。
这在你需要表示包含很多反斜杠的字符串时特别有用，比如文件路径、正则表达式等。在原始字符串中，你不需要对每个反斜杠进行转义，这可以使得代码更清晰，更易于阅读。
print(r"D:\tools\Study\tools\pear")
print("D:\\\\tools\Study\\\tools\pear")



#### 6. \b 退格符

Backspace键



#### 7. \ 续行符

当一行过长时，用于转到下一行继续输出
