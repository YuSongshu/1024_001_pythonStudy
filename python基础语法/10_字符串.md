## 索引

格式：存储字符串的变量名[下标值]，注意：从0开始

### 1.正向索引

char2 = 'abc'
print(char2[2])

### 2.负向索引
print(char2[-2])

### 3.索引超出范围

报错：IndexError: string index out of range
print(char2[-5])
print(char2[6])

## 切片

格式：[起始值：结束值：步长]
步长默认为1，结束值不包括(包前不包后)，[起始值，结束值)
range(起始值，结束值，步长)

### 正向切片

char2 = 'hello'
print(char2[0:3])
print(char2[0:3:2])

### 负向切片
print(char2[-1:-4:-1])
print(char2[-1:-4:-2])

## 成员变量 in  not in

in判断在，输出True
not in 判断不在，输出True

```
char2 = 'hello'
if 't' not in char2:
    print(f'char2中不存在字母t')
```



## 分隔符.split()

1.默认会把空白字符作为分割点

```
s = 'today the weather is sunny! to study python!'
print(s.split())
```

输出：

```
['today', 'the', 'weather', 'is', 'sunny!', 'to', 'study', 'python!']

```

2.指定分割位置

```
s2 = '2024:10:21'
print(s2.split(':'))
```

3.限制分割次数

```
s3 = '人生，苦短，我学，python！'
print(s3.split('，', 1))
```

## 字符的转换

### 转成大写字母.upper()

```
s = 'hello world'
print(s.upper())
```

输出：

```
HELLO WORLD
```



### 转成小写字母.lower()

```
s = 'WEATHER'
print(s.lower())
```

输出：

```
weather
```



### 首字母大写.title()

```
s = 'today the weather is sunny!'
print(s.title())
```

输出：

```

```



### 字符的替换.replace()

## 字符的判断

判断成功--True
判断失败--False

### .startswitch()--判断字符开头

```
char = 'the Weather is windy, to study python !'
print(char.startswith('s'))
print(char.startswith('t'))
print(char.startswith('T'))
```

输出：

```
False
True
False
```

注：

严格区分大小写

### .endswitch()--判断字符结束
```
char = 'the Weather is windy, to study python !'
print(char.endswith('!'))	#英文
print(char.endswith('！'))	#中文
```

输出：

```
True
False
```

### 应用

用户注册页面
用户名要求：纯字母，不允许有空格，前后空格自行删除，字符中间空格，提醒用户不合法

```
name = input('请输入你的用户名：').strip()   #不允许有空格，前后空格自行删除
if name.isalpha():   #纯字母True
    print(f'用户名{name}注册成功')
else:   #False,不是纯字母
    print('用户名输入不符合规范！')  # 字符中间空格，也不符合纯字母
```
