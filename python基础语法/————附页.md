## 常用函数，方法及对应的应用

### 随机数获取与猜测

```python
import random

num = random.randint(1, 100)  # [1,10] 取前也取后
difficult = int(input("难度（1.简易 2.中等 3.困难）[输入序号]："))
if difficult == 1:
    a = 9
if difficult == 2:
    a = 6
if difficult == 3:
    a = 3
for i in range(a):
    user_num = int(input('请输入数字[1-100]：'))
    if user_num == num:
        print('恭喜你猜对了')
        break
    elif user_num > num:
        print('猜的数字偏大')
    else:
        print('猜的数字偏小')
print('游戏结束')


```



















## 小代码

### 九九乘法表

```python
hang = 1
while hang <= 9:
    lie = 1
    while lie <= hang:
        print(f'{hang}*{lie}={hang * lie} ', end=' ')
        lie += 1
    hang += 1
    print()
```
