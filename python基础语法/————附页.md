## 常用函数，方法及对应的应用

### 随机数获取与猜测

```python
import random

num = random.randint(1, 100)  # [1,10] 取前也取后
difficult = int(input("难度（1.简易 2.中等 3.困难）[输入序号]："))
if difficult == 1:
    a = 9
if difficult == 2:
    a = 6
if difficult == 3:
    a = 3
for i in range(a):
    user_num = int(input('请输入数字[1-100]：'))
    if user_num == num:
        print('恭喜你猜对了')
        break
    elif user_num > num:
        print('猜的数字偏大')
    else:
        print('猜的数字偏小')
print('游戏结束')


```



## PyCharm Python 必备快捷键

一、导包 / 修复

- 自动导包、快速修复：Alt + Enter
- 优化导入（删无用包）：Ctrl + Alt + O

二、运行 / 调试

- 运行当前文件：Shift + F10
- 调试：Shift + F9
- 停止运行：Ctrl + F2
- 单行调试（下一步）：F8

三、代码快速编写

- 代码补全：Ctrl + 空格
- 快速格式化代码：Ctrl + Alt + L
- 复制一行：Ctrl + D
- 删除一行：Ctrl + Y
- 上下移动行：Alt + Shift + 上/下
- 注释/取消注释：Ctrl + /

四、查找 / 跳转

- 全局查找文件：Double Shift
- 查找内容：Ctrl + F
- 替换：Ctrl + R
- 快速跳转到定义：Ctrl + 左键
- 查找类/方法：Ctrl + Alt + Shift + N

五、重构 / 高效

- 重命名：Shift + F6
- 抽取方法：Ctrl + Alt + M
- 快速包裹代码（if/for/try）：Ctrl + Alt + T

六、界面操作

- 快速打开设置：Ctrl + Alt + S
- 隐藏/打开侧边栏：Alt + 1
- 快速回到编辑区：Esc



## 小代码

### 九九乘法表

```python
hang = 1
while hang <= 9:
    lie = 1
    while lie <= hang:
        print(f'{hang}*{lie}={hang * lie} ', end=' ')
        lie += 1
    hang += 1
    print()
```
