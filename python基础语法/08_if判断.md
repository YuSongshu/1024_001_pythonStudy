## 1. 单独的if语句
### 1.1 含义
if 语句是Python中实现条件判断的关键工具。



### 1.2 语法格式

if 条件表达式:

条件成立时执行的代码块



应用：检查温度是否高于20度，满足条件则输出穿衣建议。

```python
temperature = 15		#定义变量保存温度
if temperature > 20:
# 如果条件为真(即温度大于20度)，则执行这个代码块，否则不执行
	print("天气较热，请穿凉爽的衣服~")
print("哈哈哈")    # 没有缩进  和if语句无关
```



### 1.3 作用
if语句允许程序在满足特定条件时执行特定的代码块，如果条件不满足(即条件表达式的结果为False)，则程序会跳过该代码块，继续执行if语句之后的代码。



### 1.4 注意点

缩进：四个空格/Tab补全；
缩进后的代码，缩进位置要一致；
判断条件执行之后，自动转换成布尔值





## 2.if-else结构 （二选一）

1. 语法格式
   if 条件表达式:
   ​    条件成立时执行的代码块
   else:
   ​    条件不成立时执行的代码块



应用:检查一个人的年龄并根据其是否成年来输出相应的消息

```python
age = 16
if age >= 18:				#检查年龄是否大于或等于18岁
	print("你已经成年了.")	#如果条件为True(即年龄大于或等于18岁),则执行这个代码块
else:
	print("你还未成年呢.")	#如果条件为False(即年龄小于18岁),则执行这个代码块
```





## 3. 三元表达式

### 3.1 作用

通过允许在一行代码中完成条件判断和值的选择,显著的减少了需要编写的代码量,这种简洁性使得代码更加紧凑,减少了冗余,并且提高了可读性.当处理简单的条件逻辑时,使用三元表达式可以使代码更加清晰明了.



### 3.2 格式

为真结果 if 条件 else 为假结果



应用:检查一个人的年龄并根据其是否成年来输出相应的消息

```python
age = 16
print("你已经成年了.") if age >= 18 else print("你还未成年呢.")
print("你已经成年了." if age >= 18 else "你还未成年呢.")
```

注意:当条件逻辑复杂\需要执行多个语句\嵌套的三元表达式时,更推荐使用if-else或其他结构.





## 4. if-elif    (多选一)

### 4.1 格式

if 条件表达式1:
​    条件1成立时执行的代码块
elif 条件表达式2:
​    条件2成立时执行的代码块
elif 条件表达式3:
​    条件3成立时执行的代码块
...  		# elif没有数量限制,可以根据需要添加多个elif语句来检查多个条件
else:	#else可以不用，但若上述条件中都未通过则什么都不会输出
​    所有条件都不成立时执行的代码块



应用:检查天气情况,并根据结果输出相应的穿衣建议

```python
# # 定义变量保存天气情况
weather = input("请输入天气:")
if weather == "晴天":
# 如果天气为"晴天",则执行这个代码块
   	print("今天阳光明媚,建议穿短袖和短裤.")
elif weather == "多云天":
# 如果天气为"多云天",则执行这个代码块
	print("今天多云,建议穿轻便的外套和长裤.")
elif weather == "雨天":
# 如果天气为"雨天",则执行这个代码块
   	print("今天下雨,请记得带伞,并穿上防水的外套和鞋子.")
else:
# 如果天气不是上述中的任何一种,则执行代码块
	print("今天的天气情况不在预设范围之内,请根据实际情况选择合适的衣物.")
```



### 4.2 注意

1. if-elif-else结构按顺序在多个条件中逐一检查，一旦某个条件满足(即为True)，则立即执行该条件下对应的代码块，并跳过所有后续的 elif 和 else 及其对应的代码块。

2. 如果所有的条件都不满足,则直接执行else下的代码块.

3. if多分支，if一定是第一个分支,后续其它分支为elif

4. else不是必须的，若有则必须放在最后

5. 如果多分支中，同时满足多个判断条件，执行第一个判断的结果，不会继续向后执行



## 5. if嵌套

### 5.1 含义

指在一个if语句的代码块内部再包含另一个或多个if语句.



### 5.2 应用场景

在之前条件满足的前提下,再增加额外的判断.

应用: 首先判断用户是否有票,没票就打道回府,有票就判断安检是否通过,通过则可以上车回家,否则就抓走隔离.
是否有票:

有票
​	安检:

​		体温正常:上车回家

​		体温异常:抓走隔离

没票:打道回府



```python
#定义布尔型变量保存是否有票:True表示有票,False表示没票
has_ticket = True
temperature = 38.5			# 定义变量保存体温
if has_ticket:				# 检查是否有票
    """有票"""
    print("有票,可以安检~")
    if 36.3 <= temperature <= 37.2:		# 判断安检是否通过(体温是否正常)
        """体温正常"""
        print('体温正常,可以上车回家了~')
    else:
        """体温异常"""
        print("体温异常,抓走隔离~")
else:
    print("没票,打道回府~")
```





## 6. pass占位符

作用：暂时替代代码块，避免出现语法问题报错
使用场景：暂时没有想好代码块怎么实现，可以利用pass先搭建框架

```python
age = 16
if age >= 18:				#检查年龄是否大于或等于18岁
	pass					#只有未成年才输出，成年什么都不会输出
else:
	print("你还未成年呢.")
```
