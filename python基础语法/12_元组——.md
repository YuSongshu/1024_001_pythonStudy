# 12_元组——

元组-不可变数据类型，存储各种类型的数据 <class 'tuple'>

格式：变量名=()

```
tup = (21, 85, 'hello', [11, 'python'], ('a', 'b', 8+2j))
print(tup, type(tup))
```

## 不可变数据类型

因为不可变数据类型所以不能对元组里的数据进行增加、修改、删除

## 作用

一个元组可以存储多个数据，元组内的数据是不能修改的

函数多返回值，函数想返回多个东西，必须用元组

## 注

如果定义的元组只有一个数据,那么这个数据后面也要添加逗号，否则数据类型为唯一的这个数据的数据类型。

## 查找元素对应的索引值index()

index(元素值)

```
tup = (21, 85, 'hello', [11, 'python'], ('a', 'b', 8+2j))
print(tup.index('hello'))
```

输出：

```
2
```



## 基础操作

索引、切片、成员运算符、计数count()、长度len()			#类似列表

```
tup = (21, 85, 'hello', [11, 'python'], ('a', 'b', 8+2j), 21)
# 索引
print(tup[-2][-2])
# 切片[start:end:step]
# print(tup[0:4])
# 元组里有可变元素，可对可变元素进行修改(增删改查)
tup[3][0] = '学习'
print(tup)
# 成员运算符  in  not in
print([11, 'python'] in tup)
print([11, 'python'] not in tup)

# 计数count()
print(tup.count(21))
# 长度len()
print(len(tup))
```

输出：

```
2
b
(21, 85, 'hello', ['学习', 'python'], ('a', 'b', (8+2j)), 21)
False
True
2
6
```

注：Python 打印复数时，**自动加括号**
